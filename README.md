Students Management System
A simple web application built with Django designed to manage and display student information through a costomized administrative dashboard.

Features
 Student CRUD: Create, Read, Update and Delete student records.
 Automated Tracking: Automatically records the date and time each student entry is added.

Tech Stack
 Python 3.14
 Django 5.2
 SQLite
